## ğŸ“– How to Use

### Selecting a RiwÄyah
1. Tap the **RiwÄyah indicator** in the top-right corner of the home screen
2. Browse through the 20 available RiwÄyÄt
3. **Tap to select** any RiwÄyah:
   - If you're **online**: Stream instantly without downloading
   - If you're **offline**: Only downloaded RiwÄyÄt are available
4. To download for offline use, use the download button in the RiwÄyah selection screen

### Reading the Quran
- **Swipe right-to-left** to turn pages (Arabic reading direction)
- **Tap the screen** to show/hide controls
- **Tap bookmark icon** to save a page
- **Tap fullscreen icon** for distraction-free reading

### Using Audio Features
1. **Play the page**: Tap the audio button to hear all ayahs on the current page
2. **Play a specific ayah**: Tap on any ayah text to play just that ayah
3. **Adjust speed**: Use the speed control in the audio player
4. **Loop ayahs**: Enable looping for memorization practice

### Navigation
- **Page indicator**: Shows current page number; tap to jump to any page
- **Surah list**: Access from the home screen to jump to any surah
- **Bookmarks**: View saved pages from the home screen
