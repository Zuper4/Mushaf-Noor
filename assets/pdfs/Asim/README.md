# Asim القاري عاصم

Place the PDF files for Imam Asim's recitation here:

## Rawis (Narrators):
- **`Hafs.pdf`** - Hafs 'an 'Asim (حفص عن عاصم) - Most widely used recitation
- **`Shu3bah.pdf`** - Shu'bah 'an 'Asim (شعبة عن عاصم)

## File Requirements:
- **Pages**: 604 pages (standard Mushaf)
- **Quality**: High resolution (recommended: 300 DPI or higher)  
- **Format**: Standard PDF
- **Size**: Optimized for mobile (recommended: under 50MB if possible)

## Notes:
- Hafs is the default and most common recitation worldwide
- Both files should contain the complete Mushaf (604 pages)
- Ensure PDF quality is high for mobile viewing
- The app will automatically detect and load these files
