# Nafi' القاري نافع

Place the PDF files for Imam Nafi' from Madinah here:

## Rawis (Narrators):
- **`Qaloon.pdf`** - Qaloon 'an Nafi' (قالون عن نافع)
- **`Warsh.pdf`** - Warsh 'an Nafi' (ورش عن نافع) - Common in North Africa

## File Requirements:
- **Pages**: 604 pages (standard Mushaf)
- **Quality**: High resolution for mobile viewing
- **Format**: Standard PDF

## Notes:
- Warsh is widely used in North and West Africa
- Both recitations may have different page layouts than Hafs
