# App Icons

Place your app icon here:
- **app_icon.png** - Your main app icon (1024x1024 pixels recommended)

## Requirements for app_icon.png:
- Format: PNG
- Size: 1024x1024 pixels (minimum 512x512)
- Background: Can be transparent or solid
- Content: Should look good when scaled down to small sizes (16x16, 24x24, etc.)

After placing your icon file here, run:
```bash
flutter pub run flutter_launcher_icons
```
